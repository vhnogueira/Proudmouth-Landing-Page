# Design Guide

## Typography
- **Font Family**: Montserrat
- **Main Heading**: Large, bold, dark gray/charcoal color
  - Example: "Get Started With ProudMouth"
- **Section Headings**: Medium-large, bold
  - Examples: "Top Advisor Marketing Podcast", "Be Your Own Loud Podcast", "PodRocket Academy", "Blog"
- **Body Text**: Regular weight, dark gray, centered in card sections
- **Navigation**: White text on red background, uppercase or regular case

## Colors

### Primary Palette
- **Main Red**: #ec202c
  - Used for: Primary buttons, header background, CTAs, accent underlines
- **Dark Navy/Charcoal**: #273249
  - Used for: Footer background, text headings, dark sections

### Secondary Palette
- **Light Blue/Mint**: #bad7d9 Light cyan-blue (appears in lower section background)
- **White**: #FFFFFF
  - Used for: Card backgrounds, text on dark backgrounds, main content background
- **Light Gray**: #F5F6F6
  - Used for: Card borders, subtle backgrounds

## Components

### Header/Navigation
- **Background**: Red gradient (left to right, slightly diagonal)
- **Text Color**: White
- **Logo**: White "PROUDMOUTH" text, bold, all caps
- **Navigation Links**: White text, inline horizontal layout
- **Search Icon**: White, right-aligned

### Cards
- **Background**: White
- **Border**: 3px solid #F5F6F6 (or subtle drop shadow)
- **Padding**: 20px (generous internal padding ~40-60px)
- **Border Radius**: 0px (zero radius) - sharp corners
- **Layout**: 2-column grid on desktop
- **Icon**: Red outline icon centered at top of each card
- **Title**: Bold, centered, dark text
- **Description**: Regular weight, centered, gray text
- **Spacing**: Even gaps between cards

### Buttons
- **Primary Style**:
  - Background: Red (#ec202c)
  - Text: White, uppercase
  - Border Radius: Fully rounded (pill shape)
  - Padding: Medium (approximately 12-16px vertical, 24-32px horizontal)
  - Hover: Likely darker red or slight scale
  - Examples: "LISTEN TO THE TAM PODCAST", "LISTEN TO THE BYOL PODCAST", "LEARN MORE", "READ THE BLOG"

### Icons
- **Style**: Line art/outline style
- **Color**: Red (#ec202c)
- **Size**: Medium, approximately 60-80px
- **Examples**: Microphone in phone, graduation cap, notepad

### Dividers
- **Accent Underline**: Short red horizontal line under main heading
- **Length**: Approximately 60-80px wide
- **Color**: Red (#ec202c)
- **Thickness**: 3-4px
- **Position**: Centered below heading

### Footer
- **Background**: Dark navy (#273249 or similar)
- **Text Color**: Light gray/white
- **Layout**: Multi-column (4 columns visible)
- **Column Titles**: Slightly brighter text
- **Links**: Standard link styling, lighter gray
- **Social Icons**: White icons in row
- **Copyright**: Small text at bottom

## Layout

### Section Spacing
- **Desktop Padding**: 80px (top and bottom of sections)
- **Container Max-Width**: Centered, with side margins
- **Card Grid**: 2 columns on desktop with equal gaps
- **Content Alignment**: Centered text within cards

### Background Sections
- **White Section**: Main content area with cards
- **Light Blue/Mint Section**: Alternate background for variety, contains PodRocket Academy info
- **Dark Navy Section**: Footer area

## Design Patterns

### Visual Hierarchy
1. Large centered heading with red underline accent
2. Descriptive subheading/intro text
3. Grid of content cards with icons
4. Strong CTAs with red pill buttons
5. Alternating background colors for visual interest

### Consistency Elements
- All buttons use same pill shape style
- All section headings use similar sizing and weight
- Icons consistently use red outline style
- Generous whitespace throughout
- Centered alignment for main content sections
